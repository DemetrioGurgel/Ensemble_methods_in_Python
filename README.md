# Ensemble Methods in Python

Este repositório reúne três projetos de Machine Learning focados na aplicação prática de **modelos de ensemble** (combinação de classificadores e regressores) utilizando Python e scikit-learn

## Projetos incluídos

### 1. Predicting Movie Rental Durations
**Objetivo:**  
Prever a duração de aluguel de filmes usando modelos de regressão.

**Modelos utilizados:**
- Regressão Linear
- Random Forest Regressor

**Ensemble:**
- Combinação por média das previsões

**Métricas utilizadas:**
- Mean Squared Error (MSE)

---

### 2. What Makes a Good Book?
**Objetivo:**  
Classificar livros como populares ou não populares a partir de informações estruturais e textuais.

**Modelos utilizados:**
- Regressão Logística
- Random Forest Classifier
- Gradient Boosting Classifier

**Ensemble:**
- Soft Voting
- Ajuste de threshold

**Métricas utilizadas:**
- Accuracy
- Precision
- Recall
- F1-score

---

### 3. Facial Recognition with Supervised Learning
**Objetivo:**  
Classificar imagens como "Arnie" ou "Non-Arnie" com base em features extraídas.

**Modelos utilizados:**
- Regressão Logística (baseline)
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)

**Ensemble:**
- Soft Voting (ponderado)
- Stacking com Regressão Logística como meta-modelo

**Métricas utilizadas:**
- Accuracy
- Precision
- Recall
- F1-score
- ROC AUC
